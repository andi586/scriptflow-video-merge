# Font Files

Place the following font file in this directory:

**Required:** `Inter-Regular.ttf`

Download from: https://fonts.google.com/specimen/Inter
- Click "Download family"
- Extract the zip
- Copy `Inter-Regular.ttf` (or `Inter_18pt-Regular.ttf` from the newer package) into this folder

The end card generator uses this font for the closing title card appended to each episode.
